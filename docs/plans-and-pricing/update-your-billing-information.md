# 更新计费信息

{% hint style="success" %}
对应的[官方文档地址](https://bitwarden.com/help/article/update-billing-info/)
{% endhint %}

您的计费信息仅能通过网页 App 进行更新。

{% hint style="danger" %}
个人版方案的订阅和组织版方案的订阅是**分开**的，您需要在不同的区域更新计费信息。如果您**同时**拥有高级版和组织版订阅，为其中一个更新计费信息并不会同时更新另一个的。
{% endhint %}

## 更新个人计费信息 <a href="#update-billing-information-for-individuals" id="update-billing-information-for-individuals"></a>

要更新您的个人订阅的计费信息，请转到**设置** → **订阅** → **付款详细信息**：

<div align="left" data-with-frame="true"><figure><img src="https://bitwarden.com/assets/1VADjEEgyXqp4RxGTJE5pn/be179fbba8bb435accfb26c1f0c75ea3/Premium_payment_method.png?w=1264&#x26;fm=avif" alt=""><figcaption><p>高级版付款方式</p></figcaption></figure></div>

在此页面上，您可以：

* **添加信用额度**（PayPal 或 Bitcoin），信用额度可用于购买。可用的额度将自动用来抵扣此账户下的账单。
* **更改付款方式**（信用卡或 PayPal）。
* **添加付款方式**（信用卡或 PayPal）。

选择**计费历史记录**选项卡以：

* 查看和下载**账单**。
* 查看**交易记录**。

## 更新组织计费信息 <a href="#update-billing-information-for-organizations" id="update-billing-information-for-organizations"></a>

您必须是所有者才能更新组织的计费信息。

要更新您组织订阅的计费信息，包括其[增值税 (VAT) 或商品和服务税 (GST) 号码](tax-calculation.md#vat)（如果适用）：

1、在 Bitwarden 网页 App 中，使用产品切换器打开管理控制台：

<div align="left" data-with-frame="true"><figure><img src="https://bitwarden.com/assets/2uxBDdQa6lu0IgIEfcwMPP/e3de3361749b6496155e25edcfdcf08b/2024-12-02_11-19-56.png?w=1013&#x26;fm=avif" alt=""><figcaption><p>产品切换器</p></figcaption></figure></div>

2、从导航栏选择**计费** → **付款详细信息**：

<div align="left" data-with-frame="true"><figure><img src="https://bitwarden.com/assets/1ueaSUjVlMBXtjHroCB4Kw/38a6da67cb49413087787b57d62a2b2a/Organization_payment_method.png?w=1258&#x26;fm=avif" alt=""><figcaption><p>组织付款详细信息</p></figcaption></figure></div>

此界面为您提供以下选项：

* **更改付款方式**（信用卡或 PayPal）。
* **编辑计费地址**。
* **添加信用额度**（PayPal 或 Bitcoin），信用额度可用于购买。可用的额度将自动用来抵扣此账户下的账单。

{% hint style="info" %}
如果您的计费信息已经更新，但您的订阅仍不可用，请[联系我们](https://bitwarden.com/contact)。
{% endhint %}

## 更新组织的计费电子邮箱 <a href="#update-billing-email-for-organizations" id="update-billing-email-for-organizations"></a>

您必须是所有者，才能更新您组织的计费电子邮箱。要更新组织订阅的计费电子邮箱：

1、在 Bitwarden 网页 App 中，使用产品切换器打开管理控制台：

<div align="left" data-with-frame="true"><figure><img src="https://bitwarden.com/assets/2uxBDdQa6lu0IgIEfcwMPP/e3de3361749b6496155e25edcfdcf08b/2024-12-02_11-19-56.png?w=1013&#x26;fm=avif" alt=""><figcaption><p>产品切换器</p></figcaption></figure></div>

2、从导航栏选择**设置** → **组织信息**。

3、更新您的**计费电子邮箱**然后选择**保存**。
