## 评估：需求、评估、决策 <a href="#evaluation-requirements-assessment-decision" id="evaluation-requirements-assessment-decision"></a>

对于正在积极比较、评估和试用解决方案的团队，以下资源将帮助您了解 Bitwarden 如何满足您的需求。

**设定要求**

* [企业密码管理的关键能力](https://bitwarden.com/resources/critical-capabilities-for-enterprise-password-management/)：探索那些使企业级密码管理器与众不同的必备功能，从安全的凭据共享和基于角色的访问控制到与 SSO、SCIM 配置和审计日志的集成。
* [面向全球组织的密码管理](https://bitwarden.com/resources/password-management-for-global-organizations/)：了解 Bitwarden 如何通过多语言界面、合规性及安全访问，为分布在全球各地的员工提供支持，特别适合大规模运营的跨国团队。
* [凭据生命周期始于集中化所有权](https://bitwarden.com/resources/credential-lifecycle-management-security-perspective/#how-bitwarden-supports-credential-lifecycle-management/)：理解为什么对凭据的集中控制是安全的基础。了解 Bitwarden 如何帮助 IT 和安全团队在整个生命周期中管理凭据的创建、存储、访问、轮换和撤销。

**进行评估**

* [为生产环境准备试用](https://bitwarden.com/help/prepare-your-org-for-prod/)：充分利用您的 Bitwarden 试用。本资源将指导您完成关键设置步骤和注意事项，确保您的试用环境能够真实反映生产环境。
* [企业试用指南](https://bitwarden.com/resources/enterprise-trial-guide/)：逐步指导您规划、配置和评估 Bitwarden 的成功企业试用。

**决策**

* [InfoTech Data Quadrant 报告](https://bitwarden.com/go/password-management-data-quadrant-report/)：基于 IT 领导者的真实用户反馈，在第三方密码管理供应商对比中，了解 Bitwarden 在关键满意度和功能类别中的排名。
* [G2 Grid 报告](https://bitwarden.com/blog/bitwarden-g2-enterprise-grid/)：Bitwarden 在 G2 Grid 报告中名列前茅，体现了高满意度、易用性和市场影响力。通过对比同行评价与评分，了解 Bitwarden 与竞争对手的优劣。
* [说服您的领导团队](https://bitwarden.com/resources/bitwarden-business-insights-report/)：基于跨行业安全与 IT 专业人士的企业客户调研结果，用密码相关风险、用户行为及 IT 优先级数据武装您的管理层利益相关者。
* [建立全公司范围的共识](https://bitwarden.com/resources/successful-top-down-approach-with-your-password-manager/)：了解注重安全的组织如何从高层开始，获得高管支持并倡导安全优先的文化。

## 入职：设置、推广与培训 <a href="#onboarding-setup-rollout-and-training" id="onboarding-setup-rollout-and-training"></a>

恭喜！您已选择 Bitwarden 作为您的密码管理器。现在，是时候将 Bitwarden 集成到您现有的系统中，培训您的团队，为成功部署做准备，并向公司全体用户推广密码管理了。

**技术基础搭建**

* [个性化设置指南](https://setup.bitwarden.com/)：根据您的部署模式、组织规模和技术要求，生成定制化实施路线图的交互式工具，确保您从第一天起就聚焦正确优先级。
* [企业 Password manager 实施指南](https://bitwarden.com/resources/bitwarden-enterprise-password-manager-implementation-guide/)：关于如何在您的团队中成功启动 Bitwarden Password Manager 的分步说明。
* [云端设置清单](../../self-hosting/cloud-setup-checklist.md)：一份指导您的团队在云端设置 Bitwarden 的清单。涵盖从账户创建到策略配置的关键步骤。
* [自托管设置清单](../../self-hosting/plan-for-deployment/self-host-checklist.md)：一份确保成功部署自托管的检查清单，包括基础设施要求和安装步骤。

**准备推广**

* [Bitwarden 入职手册](bitwarden-onboarding-playbook.md)：一份面向 IT 管理员的全面 5 阶段实施路线图，包含可操作的检查清单和分步建议，以确保在您的组织中成功实现用户采用。
* [公告幻灯片模板](https://docs.google.com/presentation/d/1zK8NDB6E8ID_ok_yxn5x5qjO7mzeI5CZ-kqcOsfcQcU/edit?slide=id.g37260cb3e91_1_0#slide=id.g37260cb3e91_1_0)：用于向整个组织通告 Bitwarden Password Manager 的幻灯片模板。
* [欢迎电子邮件模板](welcome-email-templates.md)：用于向最终用户、管理员用户和 IT 团队通告 Bitwarden Password Manager 推广的电子邮件模板。
* [管理团队入职电子邮件](admin-team-onboarding-emails.md)：自动发送给新企业管理员和所有者的入职电子邮件。可一次性阅读全部内容，或直接取用并根据团队需求进行调整。
* [入职工作流程](onboarding-workflows.md)：针对不同组织设置的跨用户入职指导，无论您使用 SSO、SCIM、手动配置还是目录同步。

**培训与持续支持**

* [Bitwarden 新闻通讯](https://bitwarden.com/newsletter-subscribe/)：随时了解 Bitwarden 的最新动态。
* [产品更新](../../release-notes.md)：获取产品增强功能和新能力的发布动态。
